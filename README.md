# MPMP_Prime_Pairs_Puzzle
Bruteforcy solution to Matt Parkers Prime Pairs Puzzle,
which can be found here:
 https://www.youtube.com/watch?v=AXfl_e33Gt4

As the primes are created based on the nums-list,
changing the range in the nums-list creation is all that needs to be done,
to freely vary the puzzle parameters.
